## React blog with Node, Redis, Mongodb, Jest, Puppeteer
### Setup
For instructions on how to install nvm visit [creationix](https://github.com/creationix/nvm/blob/master/README.md)
```sh
nvm use
```
### Install dependencies
```sh
npm install
cd client
npm install
cd ..
```
### Development
```sh
npm run dev
```
### Build

```sh
npm run build
```
### Test
```sh
npm run test
```
